# dotnet-fun-hyf

.NET Introduction by HackYourFuture-Copenhagen

## Part 01 - Web API

```sh
> mkdir simple-webapi
> cd simple-webapi
> dotnet new webapi
> dotnet build
> dotnet run
> Browse https://localhost:5001/api/values
```

## Part 02 - Portal

```sh
> cd ..
> create-react-app simple-portal
> cd simple-portal
> yarn start
> Browse http://localhost:3000/
```
